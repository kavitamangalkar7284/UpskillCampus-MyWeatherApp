# 🌦️ Weather App (Java)

## Description
The Weather App is a Java-based application that allows users to check real-time weather information for any city. It fetches data from a weather API and displays details such as temperature, humidity, and weather conditions in a simple and user-friendly format.

---

## Features
- Search weather by city name
- Displays temperature (°C/°F)
-  Shows humidity levels
- Weather condition description (clear, cloudy, rain, etc.)
   Real-time data using API
- Simple console/GUI interface 
---

## Technologies Used
- Java
- REST API (e.g., OpenWeather API)
- JSON Parsing (org.json / Gson)
- HTTP Connection (HttpURLConnection)
